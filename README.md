# Коммунисты против… Плесени! v18 / V 1.2.0

HTML5 Canvas RPG-платформер-прототип.

## Запуск

- Локально для игры и ручной проверки: `npm.cmd run serve`
- Потом открой `http://127.0.0.1:4173`
- Прямое открытие `index.html` допустимо только для грубого просмотра, основной режим проверки должен идти через локальный HTTP

## Управление

- `A/D` или `←/→` — движение
- `W / ↑` — прыжок
- `S / ↓ / Ы` — спуск через одностороннюю платформу / присесть
- `ЛКМ / J / О` — атака
- `E / У` — открыть магазин / сундук / выкачивать время из врага
- `Shift` — турбо
- `F / А` — остановка времени
- `Q / Й` — сменить оружие
- `1–6` — прямой выбор оружия
- `R / К` — рестарт

## В v18 / V 1.2.0

- Главное меню снова показывает постер.
- Герой больше без сигареты.
- Добавлена активная анимация героя и врагов: бег, шаг ног, прыжок, стрельба, короткая анимация смерти.
- Перекат теперь ранит и отбрасывает врага при попадании.
- Покупка патронов в магазине теперь идет с выбором типа боеприпасов.
- Спуск вниз через верхние платформы усилен и снова проходит по `S / ↓ / Ы`.
- Враги двигаются агрессивнее и меньше дергаются на месте.
- Полоса времени героя убрана из верхнего HUD и перенесена над голову, как у врагов.
- Раш сдвинут позже; `smoke_test_v17.js` снова проходит.

Подробности: `docs/DOCUMENTATION.md`.

## Что делать при каждом изменении

Обязательно каждый раз:

1. Обновить версию игры минимум в `js/game.js`, `index.html`, `package.json`, `README.md`, при необходимости в `CHANGELOG.md`.
2. Поднять локальный сервер: `npm.cmd run serve`.
3. Открыть игру в браузере по `http://127.0.0.1:4173`.
4. Проверить вручную минимум главное меню, старт игры, движение, стрельбу, магазин и одну боевую сцену.
5. Прогнать автопроверки:
   - `npm.cmd test`
   - `node smoke_test_v17.js`
