# Коммунисты против… Плесени! — документация проекта

## Структура

```text
index.html              # точка входа
css/style.css           # стили страницы
js/input.js             # клавиатура и мышь, RU/EN раскладки
js/balance.js           # цифры баланса: оружие, враги, игрок, магазин
js/assets.js            # загрузка и отрисовка ассетов через drawImage()
js/entities.js          # Entity, Bullet, Pickup, Enemy
js/player.js            # логика Феликса
js/world.js             # биомы, платформы, подземелья, спавны, портал
js/ui.js                # HUD, магазин, интро, финал
js/game.js              # основной игровой цикл
img/imported/           # ассеты из user zip, подключены в игре
data/balance.csv        # табличное описание баланса
smoke_test.js           # базовый smoke-test
smoke_test_v15.js       # smoke-test для v15
```

## Что сейчас есть

- 6 отдельных биомов с переходом через портал в конце уровня.
- На каждом биоме один обязательный босс.
- Верхние платформы, нижнее подземелье и лестницы между этажами.
- Односторонние платформы: через них можно прыгать снизу и спускаться вниз по `S / ↓ / Ы`.
- Враги привязаны к своим этажам и патрульным зонам.
- Здоровье героя — это время. Оно постепенно убывает.
- Время можно купить в магазине или выкачать у врагов удержанием `E / У` рядом с ними.
- Остановка времени замораживает врагов и вражеские пули, но расходует время героя.
- Разные типы оружия и боеприпасов: пистолет, Мосинка, пулемёт, огнемёт, шашка, обрез.
- HUD справа сверху в стиле GTA: деньги, оружие, патроны, время, XP, мини-карта биома.
- Подключены пользовательские PNG-ассеты через `drawImage()`.

## Правило разработки

Перед каждой новой версией обязательно:

1. Проверять синтаксис:

```bash
for f in js/*.js; do node --check "$f" || exit 1; done
```

2. Запускать smoke-tests:

```bash
node smoke_test.js
node smoke_test_v15.js
```

3. Проверять вручную в браузере минимум:

- ходьбу на RU/EN раскладке;
- прыжок;
- спуск через платформы вниз;
- магазин;
- стрельбу;
- выкачивание времени;
- остановку времени;
- переход между биомами;
- рестарт.

## v17 notes

### Основная дорога
Нижний уровень и кислотная земля убраны. Главная дорога теперь ровная: один длинный `ground`-платформенный слой в `js/world.js`.

### Верхние платформы
Верхние платформы имеют тип `sky`. Они односторонние:
- снизу через них можно прыгать;
- сверху можно стоять;
- `S / ↓ / Ы` сбрасывает героя только через `sky`, но не через основной пол.

### Враги
Враги создаются не ближе `x = 620`, чтобы начало локации не превращалось в пробку из плесени. Каждый враг имеет:
- `patrolMin`;
- `patrolMax`;
- `floorY`.

`World.collide()` жёстко возвращает врага на его `floorY`, если он проваливается или улетает с этажа.

### Сложность биомов
В `Game.applyBiomeScaling()` у врагов растут HP, урон, XP и деньги за убийство в зависимости от `levelIndex`.

### Боссы
У боссов есть турбо:
- активируется рядом с игроком;
- временно ускоряет босса;
- отображается красным маркером;
- настраивается в `js/entities.js`.

### Ассеты
Новые PNG-ассеты лежат в:

```text
img/sliced/
```

Подключение ассетов:
- `img/sliced/units/`
- `img/sliced/weapons/`
- `img/sliced/bosses/`
- `img/sliced/ui_tiles/`

Их связывает `js/assets.js`.

### Обязательная проверка перед выдачей новой версии

for f in js/*.js; do node --check "$f" || exit 1; done
node smoke_test_v17.js
