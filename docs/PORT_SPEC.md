# PORT SPEC — перенос в Unity

Чертёж игры **«Коммунисты против… Плесени!»** (прототип V 1.3.0) для реализации в
Unity «по спеке». Источник правды по числам — `js/balance.js` / `data/balance.csv`;
здесь — поведение, формулы и структура. Тайминги указаны в **кадрах @60 fps**
(сек = кадры/60).

---

## 0. Концепция и пилларсы

2D сайд-скроллер-шутер, тёмный гротеск, советская эстетика. Быстрый, аркадный,
реакционный геймплей. 6 биомов, в каждом босс, финал — Ленин.
Пилларсы: **темп и реакция**, **сочность боя**, **читаемость угроз (телеграфы)**.

## 1. Кор-луп

Меню → биом: бежишь вправо, отстреливаешь врагов, проходишь 1 раш-волну с
мини-целью, добиваешь босса → портал → следующий биом. Смерть → рестарт с начала
**текущего** биома (чекпойнт). После Ленина — финал.

## 2. Управление

| Действие | Клавиши |
|---|---|
| Ходьба | A/D, ←/→, Ф/В |
| Бег | Shift |
| Прыжок (+двойной) | W / ↑ / Ц |
| Спуск сквозь платформу / прицел вниз | S / ↓ / Ы |
| Перекат (i-frames) | Z / Я |
| Турбо | C / С |
| Стоп-время | F / А |
| Атака | ЛКМ / J / О |
| Взаимодействие (магазин/сундук) | E / У |
| Смена оружия | Q / Й |
| Предметы (аптечка/противогаз) | 1–6 |
| Инвентарь | I / Ш |
| Пауза | Esc |
| Рестарт (с начала биома) | R / К |

Прицел: вверх/вниз меняет угол выстрела (диагонали).

## 3. Игрок (Феликс)

- **Здоровье** = `time` (наследие темы «время»), старт/макс **140**. Само НЕ убывает.
- Хитбокс 34×58. Гравитация **0.72**, прыжок **−8.4**, второй прыжок (способность).
- Горизонталь: accel `speed=0.32`×множители, кламп vx ±`7.2×speedMul`, трение 0.84
  (на земле) / 0.76 (в стане). `runSpeed=1.95` (бег, Shift).
- **Перекат (dodge):** длительность 14, откат 58, скорость 10.5; даёт i-frames;
  при касании врага — урон 18, отброс 10. Стоимости здоровья нет.
- **Стомп (прыжок на врага сверху):** если падаешь (vy>1) и ноги над верхом врага —
  урон **30** (+15% при finalResolve), отброс 13, отскок vy=−7.6, i-frames 16,
  hit-stop 2, тряска. Боссов топтать нельзя.
- **Турбо (C):** 5 c (300 кадров), скорость ×1.55, урон ×1.38, **жжёт здоровье**
  0.12/кадр; откат 300; после — слабость 300 (скорость ×0.52, урон ×0.68). Единственный
  режим, тратящий здоровье.
- **Стоп-время (F):** 5 c (300), откат 420; замораживает врагов и вражеские пули;
  здоровье не тратит.
- **Комбо:** окно 130 кадров; пороги попаданий [1,4,8,14] → множитель урона
  [1, 1.5, 2.2, 3.5].
- **Оглушение игрока:** 5 попаданий за 240 кадров → стан 120 кадров.
- **Газ:** НЕ снимает здоровье — **замедляет** (×0.5 на 26 кадров). Противогаз
  (предмет, цифра 2): действует 3 c (180), откат 5 c (300); даёт иммунитет к газу.
- **Прокачка/XP:** старт `xpNext=70`; при уровне: `xpNext = round(xpNext×1.4 + 30)`,
  `maxTime += 12`, лечение +38, ёмкость патронов +6 (пулемёт +18, газ +10).
- Старт: деньги 18, инвентарь [пистолет, Мосина], патроны pistol 36 / rifle 20.
- Утилиты-слоты (цифры): 1 аптечка (лечит 38, макс 9), 2 противогаз.

### Способности по биомам (открываются на входе)
`abilityUnlocks[biomeIndex]`: 0 — нет · 1 двойной прыжок · 2 турбо · 3 стоп-время ·
4 мастер шашки (шашка ×1.35 урон, +отброс) · 5 финальная решимость (×1.10 урон,
по боссам ×1.15 у стомпа, перекат быстрее).

## 4. Оружие (числа — `balance.csv`)

Смена только Q. Типы: `gun` (пуля), `shotgun` (конус из pellets), `gas` (облако),
`melee` (дуга). Пуля летит до `range`, есть отдача/нокбэк. Список: Пистолет,
Винтовка Мосина, Пулемёт (ППШ), Газомёт, Шашка, Обрез. **Огнемёта нет** (убран).

Газомёт: создаёт облако (`cloudLife 190`, `radius 64`, `tick 8.5`), бьёт зоной по
врагам; дрейфует по направлению прицела.

## 5. Враги (числа — `balance.csv`)

Общий ИИ: состояния **patrol / aggro** с памятью (`memoryTimer`), реакция на звук
выстрела (сигнал с радиусом по типу оружия), поддержка союзников, авто-разблокировка
при застревании, упреждение цели при стрельбе, стрейф и удержание дистанции у
стрелков. Враг привязан к своему `floorY` и патрульной зоне; в бою зона расширяется.

**Прыгают к игроку ТОЛЬКО зомби и бегуны.** Остальные не делают боевых прыжков.

Спецповедение по типам:
- **zombie** — ближний, рывок-выпад (lunge) вблизи.
- **runner** — быстрый ближний, кувырок-уклон под огнём.
- **pistol/rifleman/gunner** — стрелки; gunner и maxim дают веер из 3 пуль.
- **gasman** — **только ближняя дистанция** (range ~155, держит ~80): подходит и
  ставит облако газа.
- **sabreur** — выпад шашкой.
- **horse** — быстрый «чарджер».
- **kamikaze** — взрыв при смерти/низком HP (радиус 100, урон 50), мигает перед взрывом.
- **shielder** — фронтальный щит: урон спереди ×0.12.
- **sniper** — дальнобой (range 640), длинный прицел-разгон (scope 80) перед выстрелом.
- **maxim** — неподвижная турель (speed 0), частый огонь (fd 16).
- **miniboss** — элита, периодический щит (поглощает 1 удар).

## 6. Боссы

По биомам: mushroomBoss, treeBoss, sandBoss, swampBoss, factoryBoss, lenin.
Общее для всех:
- **Турбо боссa:** периодически ×1.75 скорость на 150 кадров.
- **Фаза 2** при HP < 40%: скорость ×1.55, `fireDelay ×0.62`, шанс доп-выстрела;
  визуально — красная окантовка/глоу.
- **Телеграф выстрела:** кольцо-зарядка за ~10 кадров до залпа.
- Выход из биома закрыт порталом, пока босс жив.
- **Lenin — таран:** вынд-ап 34 кадра (красная стрела-телеграф, можно отойти),
  затем рывок (ramSpeed 12, 42 кадра); откат 260. План: танк-босс сделать огромным
  (¾ экрана) и медленным — **в Unity**.

## 7. Масштабирование по биому (`applyBiomeScaling`)

`levelBoost = (уровень героя на входе в биом) − 1`. Для врага (boss — свои коэф.):
```
hp    ×= 1 + biome×0.14(boss 0.18) + levelBoost×0.16(boss 0.20)
dmg   ×= 1 + biome×0.11(boss 0.14) + levelBoost×0.10(boss 0.13)
speed ×= 1 + biome×0.09(boss 0.05) + levelBoost×0.02(boss 0.015)
xp    ×= 1 + biome×0.14
money ×= 1 + biome×0.18
```

## 8. Мир и уровни

- Мир: ширина 3200 (обучение 2240), высота 700. Земля — сплошная платформа y=485,
  высота 80. Сверху — односторонние `sky`-платформы (сквозь них прыжок снизу и
  спуск по S/↓).
- Магазин снабженца у входа (x≈185): покупка оружия и патронов по типам.
- Сундуки (лут: деньги/патроны/лечение/оружие/противогаз). *TODO: опустить к земле.*
- Портал выхода в конце (x≈2960).
- **Обучение (биом 0, Лес):** подсказки-хинты, без босса, мало врагов.
- Биомы: Лес, Зима, Пустыня, Болото, Город, Мавзолей.

## 9. Волны (rush)

- **1 раш на биом**, на 6-м (Мавзолей) — **2**.
- Триггер по X: когда игрок подходит (зона ~x−420…+160) — на месте появляются
  **часы с обратным отсчётом** (~150 кадров) + предупреждение.
- По нулю отсчёта спавнится волна **только из зомби/бегунов** (без оружия),
  количество растёт от биома.
- **Мини-цель** (чередуется): «перебить волну (N)» или «продержаться 8 c». Награда:
  деньги + лечение.

## 10. Экономика

- Глобальный множитель дохода `economy.moneyMult = 0.5` (применяется к деньгам с
  убийств, сундуков, ящиков). Цены оружия/патронов — см. `balance.csv`.
- Аптечки: шанс дропа с врага `economy.medkitDropChance = 0.26`.

## 11. Предметы и пикапы

Деньги (монета), патроны (по типам), время/здоровье (+ восстановление), аптечка
(+38, в инвентарь), противогаз (предмет-тумблер 3c/5c). Разрушаемые ящики
(`KP.Crate`) — **планируется убрать**.

## 12. «Сочность» (feel) — перенести ощущение

- **Hit-stop:** тяжёлый удар ~1–2 кадра, смерть рядового 3, босса 7.
- **Тряска экрана:** на тяжёлых попаданиях/смертях/приземлении/таране.
- **Гибы:** zombie/runner/horse/sabreur/kamikaze при смерти разлетаются кусками.
- **Кровь:** солдат-стрелки при смерти забрызгивают экран (оверлей).
- **Squash-and-stretch** героя при приземлении; дульная вспышка; числа урона.

## 13. Камера

Следует за игроком: `camX = clamp(px−390, 0, worldW−1024)`,
`camY = clamp(py−330, 0, worldH−576)`. Поверх — оффсет тряски.

## 14. Ассеты

Список и промты — `docs/ASSETS_PROMPTS.md`. Имена файлов латиницей по ID.
Музыка: треш-метал (Suno), плейлист по кругу.

## 15. Рекомендации по Unity-реализации

- **Числа баланса → ScriptableObject** (WeaponSO, EnemySO, AmmoSO, BiomeSO). Перенести
  из `balance.csv` один-в-один; биом-скейл — формулами выше.
- **Враги:** один контроллер + стейт-машина (Patrol/Aggro) + компоненты-способности
  (Shooter, Leaper, Charger, Shield, Exploder, Sniper, Turret) — навешивать по типу.
- **Физика:** Rigidbody2D (или kinematic + ручная), одно-сторонние платформы —
  PlatformEffector2D. Спуск — временное отключение коллизии.
- **Стрельба/пули:** пул объектов (object pooling).
- **Камера:** Cinemachine + импульсы для тряски (Impulse). Hit-stop — через
  `Time.timeScale` кратковременно.
- **Сцены:** один геймплейный сцена + данные биома; или сцена на биом.
- **Звук/музыка:** AudioSource пул + плейлист.

### Порядок переноса (вертикальный срез → наполнение)
1. Срез: герой (ходьба/прыжок/двойной/перекат/стрельба пистолетом) + камера + 1
   платформенный уровень + 1 враг (zombie) + 1 стрелок (pistol).
2. Кор-бой: комбо, hit-stop, тряска, смерть/гибы, HP, смерть/рестарт.
3. Оружие из SO + магазин/патроны.
4. Биом-структура, портал, переходы, чекпойнты.
5. Волны с часами и мини-целью.
6. Боссы (фазы, телеграфы, таран) — начиная с одного.
7. Масштаб по биомам, прокачка/XP.
8. Контент: 6 биомов, остальные враги, баланс.
9. Полиш: меню, обучение, мобильное управление, сюжет/кат-сцены.

> Не тащить весь объём сразу — собрать играбельный срез, держать темп игры.
