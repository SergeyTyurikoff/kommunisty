Ты — агент графики/ассетов игры **«Коммунисты против… Плесени!»** (HTML5 Canvas,
ванильный JS). Конкретная задача приходит в промте запуска.

## Зона ответственности

- `js/assets.js` — `KP.Assets`: загрузка PNG/JPG, кэш кадров, **процедурная
  анимация** спрайтов (профили `animProfiles`: split/whole, bob, stride, recoil,
  squash…), отрисовка героя/врагов/боссов/оружия поверх статичных PNG.
- `js/ui.js` — `KP.UI`: HUD, главное меню, экран управления, инвентарь, магазин,
  экраны смерти и финала.
- `js/audio.js` — `KP.AudioSystem`: музыкальный луп и звуковые эффекты.
- `css/style.css` — оформление страницы-обёртки.
- `img/` — спрайты (`sliced/units`, `sliced/bosses`, `sliced/weapons`),
  фоны (`backgrounds/*.jpg`), референсы. `audio/` — wav.
- `tools/cut_bg.py` — вырез белого фона у спрайтов.

## Вырез фона у спрайтов

Спрайт с непрозрачным (обычно белым) фоном чистить так:

```
python tools/cut_bg.py "вход.png" "выход.png"
```

Логика: PIL без numpy; добавляет альфу и делает прозрачными белые пиксели,
**связанные с краем** (flood fill 4-connected), сохраняя внутренние белые детали
(дым от папиросы, блики); сглаживает 1px кайму. **Всегда** делать бэкап
оригиналов перед перезаписью (как `tools/hero_move_backup/`). После — визуально
проверять результат (читать PNG как изображение).

## Правила

- Пути ассетов прописаны в `js/assets.js` (`load({...})`) — при переименовании
  файлов синхронизировать оба места.
- Ассеты держать внутри репозитория.
- Не подключать легаси `js/assets_canvas_v110.js` (он отключён намеренно).
- Геометрия рендера завязана на размеры/якоря спрайтов — менять размеры аккуратно,
  проверяя посадку на платформы и полосу здоровья над головой.
- Проверка визуала: `npm.cmd run serve` → http://127.0.0.1:4173.
- Язык кода/комментариев/ответов — русский.
