## 1. Что это за проект

2D экшен-платформер **«Коммунисты против… Плесени!»** на чистом HTML5 Canvas
**без фреймворков и без шага сборки**. Игрок управляет революционером (Феликс),
проходит 6 биомов, в каждом — свой босс, финал — Ленин.

- Точка входа: `index.html` — подключает 10 `<script>` по порядку.
- Весь рантайм-код в `js/`, в глобальном неймспейсе `window.KP`.
- Игра стартует из главного меню, а не сразу в игровой сцене.
- Никакого транспайла/бандла: правишь файл → перезагружаешь страницу.

## 4. Игровой цикл и состояние

`KP.Game.loop()` (`js/game.js`): `update() → draw() → input.tick()` через
`requestAnimationFrame`. Формальной стейт-машины нет — состояние держится на
флагах: `ui.intro`, `paused`, `player.dead`, `ui.ending`, `levelTransition`,
`ui.shopOpen`, `ui.inventoryOpen`. Логически: главное меню → игра (по биомам с
порталами) → смерть/рестарт или финал (после Ленина).

Камера следует за игроком и клампится к границам мира (`updateCamera`).

## 5. Ключевые игровые системы

- **Здоровье («время»).** Шкала `player.time` над головой героя — это просто
  здоровье (название «время» оставлено как тема). Само не убывает; пополняется
  пикапами `time`, аптечками и уровнями. Идея «время как расходуемый ресурс»
  отменена (см. §8 и `docs/DOCUMENTATION.md`).
- **Оружие (6):** пистолет, винтовка Мосина, пулемёт, газомёт (облако-зона),
  шашка (мили), обрез (конус дроби). Огнемёт убран. Смена — **только Q**;
  цифры 1–6 — быстрые предметы (аптечка/противогаз). 5 типов патронов в `KP.Balance`.
- **Газ** только замедляет ГГ (не урон); противогаз 3 c / откат 5 c. **Стомп**:
  прыжок на врага сверху — урон+отскок без потери здоровья (кроме боссов).
  **Экономика**: `KP.Balance.economy.moneyMult`. Раш — только зомби (1/биом, 2 на 6-м).
- **Способности по биомам** (`KP.Balance.abilityUnlocks`, индекс = номер биома):
  двойной прыжок, турбо, стоп-время, мастер шашки, финальная решимость.
  - **Турбо (`C/С`)** — единственный режим, тратящий здоровье
    (`turboHealthPerFrame`), затем слабость; **стоп-время (`F/А`)** и **перекат
    (`Z/Я`)** теперь без затрат здоровья, ограничены откатом.
- **Комбо-множитель** урона за серию попаданий (`registerHit`).
- **Стан игрока:** 5 попаданий подряд → оглушение 2 сек (`registerEnemyHit`).
- **Прокачка:** XP→уровни повышают maxTime и ёмкость патронов; враги
  масштабируются по номеру биома и уровню героя (`applyBiomeScaling`).
- **Чекпойнты между биомами:** прогресс сохраняется на входе в биом
  (`saveCheckpoint`); смерть/`R` откатывают к началу биома (`restoreCheckpoint`),
  полный сброс — только из меню/финала (`fullReset`).
- **Волны (rush)**: перед волной на её месте появляются **часы** с обратным
  отсчётом (`checkRush`/`drawRushClocks`), у волны есть **мини-цель** (перебить
  всех / продержаться) с наградой (`waveObjective`).
- **«Сочность»:** hit-stop (`hitStop`) и тряска экрана (`shake`) на тяжёлых
  попаданиях и смертях; **гибы** (`KP.Gib`) — одних врагов разрывает на части,
  другие при смерти **пачкают экран кровью** (`addBlood`); squash при приземлении.
- **Магазин** (E у снабженца): покупка оружия и конкретных типов патронов;
  **сундуки** и **разрушаемые ящики** (`KP.Crate`) с лутом.

### ИИ врагов (`js/entities.js`) — самая богатая часть

Состояния `patrol` / `aggro` с памятью (`memoryTimer`), реакция на звук
выстрелов (`game.playerShotSignal`), поддержка союзников, авто-разблокировка при
застревании (`resolveStuckState`), упреждение цели при стрельбе, удержание
дистанции и стрейф у стрелков. Уникальные приёмы: рывок зомби, выпад сабельщика,
кувырок-уклон раннера, щит минибосса/щитоносца, прицел-разгон снайпера, взрыв
камикадзе, таран Ленина, **вторая фаза боссов** при HP < 40% (`_triggerPhase2`).
Враги привязаны к своему этажу (`floorY`) и патрульной зоне.

### Биомы и боссы (по индексу уровня)

## 6. Графика и ассеты

- `KP.Assets` грузит картинки и рендерит спрайты с **процедурной анимацией**
  (профили `animProfiles`: split/whole-режимы, bob, stride, recoil и т.п.) —
  поверх статичных PNG.
- База героя: `img/sliced/units/hero_revolutionary.png` (спокойные/боевые позы).
- Анимация движения: `img/sliced/units/hero_move/*.png` (run/jump/fall) **сейчас
  отключена** тумблером `useMoveSprites=false` в `_buildHeroFrames` (`js/assets.js`)
  — все состояния, включая run/jump/fall, рисуются процедурной анимацией базового
  спрайта `hero_revolutionary.png`. Чтобы вернуть отдельные PNG — поставить `true`.
- Враги/боссы: `img/sliced/units/`, `img/sliced/bosses/`.
- Оружие: `img/sliced/weapons/`. Фоны биомов: `img/backgrounds/*.jpg`.
- Платформы — нейтральный камень + слабая биомная тонировка верхней полосы.

### Вырез фона у спрайтов

Если спрайт приходит с непрозрачным (обычно белым) фоном — вырезать заливкой от
краёв, чтобы сохранить белые детали внутри (дым, блики):

```
python tools/cut_bg.py "вход.png" "выход.png"
```

Скрипт (`tools/cut_bg.py`, чистый PIL, без numpy):
- добавляет альфа-канал, делает прозрачными белые пиксели, **связанные с краем**
  изображения (flood fill 4-connected); внутренние белые детали остаются;
- сглаживает 1px белую кайму (defringe).
- Спрайты `hero_move/` уже обработаны им; оригиналы — в `tools/hero_move_backup/`.

## 7. Локальный запуск и проверки

Окружение: Windows 11, **PowerShell**, системный Node **v22.17.0**.
(В `AGENTS.md` упоминается несуществующий portable `node-v24` — использовать
системный Node.)

```powershell
# запуск
npm.cmd run serve      # tools/dev_server.js, http://127.0.0.1:4173
# (порт можно переопределить: $env:KP_PORT=5000; node tools\dev_server.js)

# smoke-тест (поднимает свой сервер на 4273, гоняет Chrome/Edge через playwright-core)
npm.cmd test           # = node tests\playtest_smoke.js
```

Перед сдачей по возможности: открыть игру, начать новую игру, сделать короткую
визуальную проверку. Smoke-тест опирается на `window.KP_TEST` (см. ниже) — не
ломать его API при правках геймплея.

### Тестовый хук `window.KP_TEST` (`js/game.js`)

`getGame`, `startGame`, `tick(frames)`, `spawnEnemy(kind,offsetX)`,

## 9. Управление

- `A/D` или `Ф/В` / стрелки — ходьба; `Shift` — бег
- `↑/W/Ц` — прыжок; `↓/S/Ы` — спуск через одностороннюю платформу / прицел вниз
- `Z/Я` — перекат (неуязвимость); `C/С` — турбо (тратит здоровье); `F/А` — стоп-время
- `ЛКМ/J/О` — атака; `E/У` — взаимодействие; `Q/Й` — смена оружия
- `1–6` — быстрые предметы (аптечка/противогаз); `I/Ш` — инвентарь
- `Esc` — пауза; `R/К` — рестарт (с начала пройденного биома)

## 10. Короткие git-команды (из AGENTS.md)

Выполнять **из папки этого репозитория**.

- `ком` — коммит: `git add …` + `git commit -m "…"`.
- `пуш` — отправить на GitHub: `git push origin master`.
- `пул` — стянуть и влить: `git pull`.

`git fetch` только забирает данные без слияния (`git log HEAD..origin/master`
для просмотра входящих), `git pull` = `fetch` + `merge/rebase`.

В конце коммит-сообщений добавлять строку:
